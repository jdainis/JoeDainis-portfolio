What Does This All Mean?

  1. The first conclusion we can come to is that using DOSE as a predictor of the Y outcome variable is quite weak. Our RMSE metric was 666 and the R-squared was 0.516, indicating that the model is not total and utter garbage, but that using dose alone to predict the outcome is not reliable.
  2. We see that including all of the variables into the linear regression model improved the metrics (smaller RMSE to 591, larger R-squared to 0.619), but still, all of the variables are not enough to strongly predict the outcome. This matches what I would expect with complicated drug data, with many variables likely impacting the outcome. This could also indicate trialing out higher doses (if possible) to see if this predictive value could get any stronger.
  3. SEX is a categorical variable that cannot be predicted by these variables (which we knew coming into this analysis).

Final Conclusions

  1. Overall, we have validated that the model using dose as a predictor of Y outcome as well as the model using all variables as predictors both outperformed our Null model. (RMSE = 948 for Null reduced to RMSE = ~600-700 depending on model used and assessment).
  2. Dose accounts for a majority of the predicting ability, as using all variables sees an additional but very slight improvement in predicting ability compared to using dose alone. We also see that the model applied to the testing data intermeshes quite nicely with the training data, indicating that this model was successful in its development.
  3. Overall, the results do make sense to me. It would make sense that a clinically approved drug in Mavoglurant used at higher doses would have an increasing detectable concentration of the drug. The dose model might be useful for assessing general trends, but it is not a perfect predictor and it would likely not be useful to use to predict on an individual case-by-case basis.
  4. Finally, for the All Variables model, as mentioned before it does see a slight improvement in predictive power, but still since it is only a slight improvement I would not use this on an individual case basis to predict an outcome. It does make sense that using all of the available variables would slightly improve predictive capability of the outcome, but this also proves that the main variable driving the predictive power of the model is dose, which makes sense for this dataset studying the pharmokinetics of the drug.
